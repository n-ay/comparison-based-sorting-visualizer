# Sorting Visualizer (For Comparison Based Sort)

## Features:
1. Generates a Random Array
2. Sorts in Speed- Fast, Medium and Slow
3. Sorting Algorithms that are visualized:
   a. Bubble Sort
   b. Insertion Sort
   c. Selection Sort
   d. Quick Sort
   e. Heap Sort
   f. Merge Sort
